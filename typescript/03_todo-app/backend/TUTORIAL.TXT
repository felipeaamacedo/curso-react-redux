Criação do backend inicia com a criação do projeto json.

npm init -y

em seguida são instalados as dependências do backend.

As dependências usadas são:

npm i --save -E body-parser@1.15.2 express@4.14.0 mongoose@4.7.0 node-restful@0.2.5 pm2@2.1.5


Outra dependências que será usada em modo de desenvolvimento é o

npm i --save-dev -E nodemon@1.11.0

PASSO2. criar uma pasta chamada "data/db"
